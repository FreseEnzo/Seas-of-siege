Detailed Description

Controls the player's movement, rotation, and combat actions within the game.

This class manages player movement on designated island blocks, handling input for movement and rotation, as well as triggering sword attacks. The player's position is validated to ensure they stay within the island bounds, and animations are triggered accordingly.
