Detailed Description

Manages the placement and switching of weapon prefabs in the scene.

This class handles the placement of weapons on an island by detecting mouse input and ensuring the weapon is correctly positioned. It also provides functionality for switching between different weapon prefabs, checking if the weapon is in water, and handling weapon placement constraints.
