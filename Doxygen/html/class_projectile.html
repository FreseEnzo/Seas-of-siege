Detailed Description

Abstract base class for projectiles in the game.

This class provides the basic functionality for all projectiles, including handling collisions with enemies and other specific objects (like water). Subclasses should implement any additional behavior specific to different types of projectiles.

virtual void Projectile.OnCollisionEnter ( Collision collision)
inlineprotectedvirtual

Handles the collision event for the projectile.

This method is triggered when the projectile collides with another object. If the object is tagged as "Enemy", it applies damage to the enemy. If the object is water or another specific layer, the projectile is destroyed.
